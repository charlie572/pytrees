# trees

Python3 implementation of trees.

## AVLTree
AVL Tree. Gurantee for balance.
- insert(self, val)
- delete(self, key)
- search(self, key)
- getDepth(self)
- preOrder(self)
- inOrder(self)
- postOrder(self)
- countNodes(self)
- buildFromList(cls, l)

## BinarySearchTree
Simple implementation of Binary Search Tree. No gurantee for balance.
- insert(self, val)
- delete(self, key)
- search(self, key)
- getDepth(self)
- preOrder(self)
- inOrder(self)
- postOrder(self)
- countNodes(self)
- buildFromList(cls, l)
